API Documentation

Welcome to the GestureBridge AI API documentation. Our API enables you to integrate real-time sign language translation capabilities into your applications.

Base URL

https://api.gesturebridge.ai/v1

Authentication

All API requests require authentication using JWT tokens.

Login

POST
/api/auth/login

Request Body

{
    "email": "user@example.com",
    "password": "your_password"
}

Response

{
    "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "token_type": "bearer",
    "expires_in": 3600
}

Streaming API

The Streaming API enables real-time sign language translation.

Start Stream

POST
/api/streaming/start

Request Body

{
    "language": "ASL",
    "quality": "high"
}

Response

{
    "session_id": "stream_123",
    "status": "ready",
    "websocket_url": "wss://..."
}

Languages API

Access and manage supported sign languages.

Get Supported Languages

GET
/api/languages

Response

{
    "languages": [
        {
            "code": "ASL",
            "name": "American Sign Language",
            "accuracy": 99.2
        },
        // ...
    ]
}

Feedback API

Submit and manage translation feedback.

Submit Feedback

POST
/api/feedback/submit

Request Body

{
    "translation_id": "123",
    "rating": 5,
    "comment": "Great translation!"
}

WebSocket API

Real-time communication for live translation.

Events

connect

Emitted when connection is established

translation_result

Received when translation is complete
